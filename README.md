You can also LIKE/DISLIKE your post.Author can aslo edit their blog. You can even view your profile and view your contribution. Tech Stack Used- CSS, JavaScript, Nodejs, ejs, MongoDB,Cloudinary,Mongoose,BootStrap.

Live Demo: https://inknest.onrender.com

Environment Variables:  
Create a .env file in the root directory and add:  
MONGODB_URL=your_mongodb_connection_string   
SESSION_SECRET=your_session_secret  
CLOUDINARY_CLOUD_NAME=your_cloudinary_cloud_name  
CLOUDINARY_KEY=your_cloudinary_api_key  
CLOUDINARY_SECRET=your_cloudinary_api_secret  

Installation & Setup (Local):

1. Clone the repository: git clone https://github.com/harshitaagoyal/InkNest.git  
2. Navigate to the project directory: cd InkNest  
3. Install dependencies: npm install  
4. Add environment variables (.env)  
5. Start the server: node app.js
6. Open browser at: http://localhost:7000
