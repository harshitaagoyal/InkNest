# InkNest- A Blogging Website 
InkNest is a blogging website where users can add his/her blogs and can comment on other’s blogs also.After
logging in, you can add your thoughts on the blog as a comment. You can also LIKE/DISLIKE your post.Author can aslo
edit their blog. You can even view your profile and view your contribution. Tech Stack Used- CSS, JavaScript, Nodejs,ejs, MongoDB,Cloudinary,Mongoose,BootStrap.
